- 10 არ არის არმსტრონგის რიცხვი რადგან 10 != 1^2 + 0^2
- 153 არის არმსტრონგის რიცხვი რადგან 153 = 1^3+ 5^3 + 3^3 = 1 + 125 + 27 = 153
